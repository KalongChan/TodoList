# Run Locally

#### Clone the project<br /> 
```git clone https://github.com/KalongChan/React-Todo.git```

#### Go to the project directory<br />
```cd Netflix-Clone```

#### Install dependencies<br />
```npm install```

#### Run the project<br />
```npm start```
